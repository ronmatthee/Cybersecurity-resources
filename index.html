Cybersecurity Resources

Tools

  • Nmap - Nmap is a free and open-source network scanner used for discovering hosts and services on a computer network.
  • Wireshark - Wireshark is a network protocol analyzer that lets you capture and interactively browse the traffic running on a computer network.
  • Metasploit - Metasploit is a penetration testing framework that enables you to find, exploit, and validate vulnerabilities in systems.
  • OpenVAS - OpenVAS is an open-source vulnerability scanner used for detecting vulnerabilities in computer systems and applications.
  • Nessus - Nessus is a vulnerability assessment tool that scans for vulnerabilities, misconfigurations, and malware on networks.
  • Snort - Snort is an open-source intrusion detection system (IDS) and intrusion prevention system (IPS) that monitors network traffic.
  • Acunetix - Acunetix is a web vulnerability scanner used to detect and manage security vulnerabilities in web applications.
  • Burp Suite - Burp Suite is a web application security testing toolkit used for scanning, crawling, and exploiting web applications.
  • OSSEC - OSSEC is an open-source host-based intrusion detection system (HIDS) used for real-time log analysis, file integrity checking, and rootkit detection.
  • Splunk - Splunk is a data analytics platform used for searching, monitoring, and analyzing machine-generated data such as logs and event data.
  • Qualys - Qualys is a cloud-based security and compliance platform used for vulnerability management, policy compliance, and web application security.
  • GnuPG - GnuPG (GNU Privacy Guard) is a free and open-source implementation of the OpenPGP encryption standard used for encrypting and signing data.
  • Sn1per - Sn1per is an automated penetration testing framework used for reconnaissance, scanning, and vulnerability assessment.
  • Shodan - Shodan is a search engine that lets you find devices connected to the internet, including servers, routers, webcams, and IoT devices.
  • Graylog - Graylog is an open-source log management platform used for collecting, indexing, and analyzing log data from various sources.
  • OWASP ZAP - OWASP ZAP (Zed Attack Proxy) is an open-source web application security scanner used for finding security vulnerabilities in web applications.
  • Bro - Bro is an open-source network security monitor used for analyzing network traffic, detecting intrusions, and generating security logs.
  • OpenVPN - OpenVPN is an open-source virtual private network (VPN) solution used for creating secure connections over the internet.
  • Tor - Tor is free and open-source software for enabling anonymous communication and browsing on the internet.
  • Vega - Vega is an open-source web security scanner and testing platform used for finding security vulnerabilities in web applications.
  • Cloudflare - Cloudflare is a web security and performance company offering services such as DDoS protection, web application firewall (WAF), and content delivery network (CDN).
  • Elasticsearch - Elasticsearch is a distributed search and analytics engine used for real-time search, logging, and analytics.
  • Cuckoo Sandbox - Cuckoo Sandbox is an open-source malware analysis system used for automated dynamic analysis of suspicious files.
  • Malwarebytes - Malwarebytes is an anti-malware software used for detecting and removing malware infections from computers.
  • Hybrid Analysis - Hybrid Analysis is a malware analysis platform that combines static and dynamic analysis techniques to analyze and detect malware.
  • Qubes OS - Qubes OS is a security-focused operating system that compartmentalizes applications into isolated virtual machines (VMs) for enhanced security.
  • OpenIOC - OpenIOC (Open Indicators of Compromise) is an open standard for sharing threat intelligence and indicators of compromise (IOCs) across security tools and platforms.
  • Endgame - Endgame is a cybersecurity platform offering endpoint detection and response (EDR), threat hunting, and automated threat intelligence.
  • Core Impact - Core Impact is a penetration testing tool used for simulating real-world attacks and testing the security posture of networks and systems.
  • Wazuh - Wazuh is an open-source security monitoring platform used for threat detection, log analysis, and compliance monitoring.
  • SolarWinds Security Event Manager - SolarWinds Security Event Manager is a comprehensive SIEM tool that provides real-time log collection, analysis, and threat detection.
  • AlienVault OSSIM - AlienVault OSSIM (Open Source Security Information and Event Management) is an open-source SIEM tool that integrates threat intelligence, log management, and vulnerability assessment.
  • osquery - osquery is an open-source endpoint security tool that allows you to query and monitor operating system telemetry in real-time using SQL-like queries.
  • Zeek - Zeek, formerly known as Bro, is an open-source network security monitoring tool used for analyzing network traffic and detecting security threats.
  • IBM QRadar - IBM QRadar is an enterprise SIEM solution that provides advanced threat detection, incident response, and security analytics capabilities.
  • Tenable.io - Tenable.io is a cloud-based vulnerability management platform that offers vulnerability assessment, prioritization, and remediation capabilities.
  • Sqreen - Sqreen is a web application security platform that provides runtime application self-protection (RASP), security monitoring, and protection against OWASP Top 10 vulnerabilities.
  • Keeper Security - Keeper Security is a password management solution that offers secure password storage, password sharing, and multi-factor authentication (MFA) capabilities.
  • Veeam - Veeam provides security hardening guides and best practices for securing virtualized and cloud environments, including VMware, Hyper-V, and AWS.
  • Veracode - Veracode is a cloud-based application security platform that offers static application security testing (SAST), dynamic application security testing (DAST), and software composition analysis (SCA).
  • Rapid7 InsightVM - Rapid7 InsightVM is a vulnerability management solution that provides visibility into your assets, prioritizes vulnerabilities, and helps you remediate security issues.
  • FireEye Helix - FireEye Helix is a security operations platform that combines SIEM, orchestration, and threat intelligence to help organizations detect, investigate, and respond to cyber threats.
  • Trend Micro Smart Protection Suite - Trend Micro Smart Protection Suite is an endpoint security solution that provides protection against malware, ransomware, and advanced threats.
  • Cisco Umbrella - Cisco Umbrella is a cloud security platform that provides DNS and IP layer security to protect users from internet threats, malware, and phishing attacks.
  • Check Point SandBlast Network - Check Point SandBlast Network is an advanced threat prevention solution that provides zero-day protection against known and unknown cyber threats.
  • Forcepoint Data Loss Prevention - Forcepoint Data Loss Prevention (DLP) is a comprehensive solution that helps organizations prevent data leaks, enforce compliance policies, and protect sensitive information.
  • Symantec Endpoint Security - Symantec Endpoint Security is an endpoint protection platform that combines antivirus, anti-malware, firewall, and intrusion prevention capabilities.
  • Palo Alto Networks Prisma Access - Palo Alto Networks Prisma Access is a secure access service edge (SASE) platform that provides secure connectivity and access control for remote users and branch offices.
  • Cybereason Endpoint Detection and Response - Cybereason Endpoint Detection and Response (EDR) is a next-generation endpoint security solution that detects and responds to advanced threats in real-time.
  • Sophos Phish Threat - Sophos Phish Threat is a phishing simulation and awareness training solution that helps organizations educate employees about phishing attacks and improve security awareness.
  • Fortinet Cloud Application Security Broker - Fortinet Cloud Application Security Broker (CASB) provides visibility, control, and protection for cloud applications and services.
  • Microsoft Defender for Endpoint - Microsoft Defender for Endpoint is an endpoint security solution that provides threat protection, endpoint detection and response (EDR), and automated investigation and remediation.
  • Varonis Data Security Platform - Varonis Data Security Platform provides data-centric security solutions for protecting sensitive data, detecting insider threats, and ensuring compliance.
  • Kaspersky Small Office Security - Kaspersky Small Office Security is a comprehensive security solution designed for small businesses, providing protection against malware, ransomware, and online threats.
  • SonicWall Firewall - SonicWall Firewall is a next-generation firewall (NGFW) that provides advanced threat protection, intrusion prevention, and secure access control for networks.
  • Qualys Cloud Platform - Qualys Cloud Platform is a cloud-based security and compliance platform that offers vulnerability management, policy compliance, and web application security solutions.
  • Zscaler Internet Access - Zscaler Internet Access is a cloud-delivered internet security service that provides secure web gateway (SWG), advanced threat protection, and data loss prevention (DLP).
  • Splunk Phantom - Splunk Phantom is a security orchestration, automation, and response (SOAR) platform that helps organizations automate and streamline security operations.
  • Forcepoint Advantage - Forcepoint Advantage is a cloud security platform that provides data protection, cloud security, and threat protection solutions for users, data, and applications.
  • CyberArk Privileged Access Security - CyberArk Privileged Access Security is a privileged access management (PAM) solution that helps organizations secure, manage, and monitor privileged accounts and credentials.
